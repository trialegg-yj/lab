(내 개인 페이지에서 시작할 수도 있어요. 개인 페이지 안에서 필터·정렬 등 보기 뷰는 자유롭게 원하는 대로 바꿔가며 사용할 수 있어요.)

진행상태를 기록해요

완료된 태스크는 반드시 체크를 표시해서 완료 처리해주세요.
세부 진행상태는 상태값으로 표시하면 돼요.

5

미팅 후 회의록은 공유일정 안에 남겨요

팀 전체가 알아야 하는 일정은 📅 공유일정에 추가해요.
회의 시작 전 녹음 버튼을 눌러 진행하고, 미팅이 끝나면 완료 여부를 체크해주세요.

6

해당 날짜 공유일정 페이지에 주간회의록이 자동으로 생성돼요.
주요 논의 내용과 액션 아이템을 기록해요. 안건이 있다면 미리 추가해둬요.

* 헷갈릴 때 이것만 기억하세요
업무는 → 🥊 프로젝트 DB  ·  일정/회의는 → 📅 공유일정 DB
05 · 팀 약속
함께 지켜요 👋
각자의 작은 습관이 모여 팀 전체를 빠르게 움직여요.
🔔
슬랙 알림은 당일 확인
빈값 알림이나 업무 알림은 당일 안에 확인하고 처리해요.
미루면 쌓이고, 쌓이면 아무도 안 보게 돼요.
📨
맥락 포함 업무 요청
요청 태스크를 만들 때 맥락·목적·목표·제약사항을 적어요.
받는 사람이 다시 물어보는 시간을 줄여줘요.
📅
회의록은 공유일정에 기록
회의가 끝나면 해당 공유일정 페이지 안에 회의록을 남겨요.
나중에 "그 회의 내용 어디 있지?" 할 때 바로 찾을 수 있어요.
📥
항목 생성 후 즉시 입력
업무를 생성하면 담당자·분야·우선순위·마감일은 그 자리에서 입력해요.
나중에 채우려다 잊으면 팀 대시보드 사용의 의미가 흐려져요.
🔗
상위 항목 연결 필수
에픽은 프로젝트에, 태스크는 에픽에 연결해서 만들어요.
연결이 없으면 진행률이 집계되지 않아요.
✅
완료 즉시 상태 업데이트
완료한 업무·일정은 즉시 완료 처리해요.
💬
개선 의견은 언제든 환영
쓰다가 불편하거나 개선이 필요하다고 느끼는 부분이 있다면 적극적으로 의견을 내주세요.
작은 디테일이 차곡차곡 쌓여 팀 전체의 시너지를 만들어요.
06 · 기능 가이드
상황별로 이렇게 써요
내 상황과 가장 가까운 케이스를 선택하세요.
예시. 출근하자마자 오늘 뭐부터 해야 할지 정리하고 싶어요.
대시보드 임베딩 뷰, 프로젝트 DB, 개인 페이지 세 곳에서 확인할 수 있어요.
1

대시보드 🥊 프로젝트 섹션 ↗ → 내 프로젝트 탭 클릭
내가 담당자인 항목만 즉시 확인할 수 있어요.

2

분야별 탭으로 전환하면 사업기획·주류사업 등 업무 카테고리 기준으로 묶어서 볼 수 있어요.

3

더 상세하게 보려면 프로젝트 바로가기 ↗ 클릭 → 내 담당 뷰
우선순위(P1→P4)·마감일 기준으로 자동 정렬된 내 전체 업무가 한눈에 보여요.

4

개인 페이지 ↗에서는 필터·정렬을 자유롭게 바꿔가며 나만의 뷰로 커스텀할 수 있어요.

예시. OO 프로젝트를 팀에서 새로 시작하기로 했어요.
큰 방향(프로젝트) → 작업 묶음(에픽) → 세부 액션(태스크) 순서로 만들면 돼요.
1

대시보드 🥊 프로젝트 섹션에서 프로젝트 추가하기 클릭
레벨이 🏆 프로젝트로 고정된 페이지가 생성돼요. [직접 진행] 업무명 템플릿이 자동 적용돼요.
맥락·목적·완료기준 작성 후 시작을 권장해요.

2

페이지 열어서 제목·분야·우선순위·마감일 입력
담당자·시작일·진행상태는 자동으로 채워져 있어요. 필요하면 변경할 수 있어요.

3

에픽 추가하기 클릭 → 예시. OO 에픽 생성
프로젝트 또는 에픽 페이지 안에서 새 하위 항목 추가를 클릭해 에픽·태스크를 만들면, 상위 항목의 진행률에 자동 반영돼요.

4

태스크 추가하기(직접 진행) 또는 태스크 추가하기(요청) 클릭
상위 항목에 에픽을 연결해 두면, 완료 시 완료여부 체크박스 하나로 에픽·프로젝트 진행률까지 한번에 올라가요.

예시. 지금 내가 할 수 없거나, 다른 팀원이 더 적합한 업무예요.
요청 태스크는 제약사항·맥락을 작성하는 섹션이 포함된 별도 템플릿으로 만들어져요.
1

태스크 추가하기(요청) 클릭
레벨이 ✅ 태스크로 고정된 페이지가 생성돼요. [요청] 업무명 템플릿에는 직접 진행과 달리 제약사항 섹션이 포함돼요.

2

템플릿에 해당하는 맥락·목적·목표·제약사항을 적어서 함께 전달하면
받는 사람이 맥락을 빠르게 파악할 수 있어요.

예시. 다음 주 외부 미팅이 잡혔어요. 팀에 공유하고 나중에 회의록도 남기고 싶어요.
모든 팀 일정과 회의록은 📅 공유일정에서 통합 관리해요.
1

대시보드 📅 공유일정 섹션 → 일정 추가 버튼 클릭
날짜·참여자·태그(외부미팅)·장소 등 기본적인 정보를 입력해둬요.

2

달력 뷰로 전환하면 팀 전체 일정을 월·주 단위로 한눈에 확인할 수 있어요.

3

미팅 당일, 회의 시작 전 녹음 버튼을 눌러 진행해요.

4

회의가 끝나면 해당 공유일정 페이지 안에 회의록 기록을 중지하고 일정 완료여부 체크
"11일 미팅 회의록 어디 있지?" → 11일 공유일정 페이지 안을 확인하면 돼요.

5

새로만들기 옆 버튼을 눌러 원하는 템플릿으로 생성할 수도 있어요.

예시. 이번 분기 프로젝트들이 어떻게 돌아가고 있는지 한눈에 보고 싶어요.
프로젝트 DB의 다양한 뷰를 목적에 맞게 골라 쓰면 돼요.
1

대시보드 타임라인 탭 → 시작일~마감일 기간으로 프로젝트·에픽 일정 흐름을 시각적으로 파악해요.

2

프로젝트 DB → 프로젝트 현황 뷰 → 프로젝트별 진행률 한눈에 확인

3

칸반보드 뷰 → 진행상태(시작전·진행중·완료·보류) 컬럼별로 프로젝트 현황을 시각화해요.

4

담당자별 뷰 → 팀원별로 어떤 업무가 얼마나 있는지 확인해요.

5

전체(빈값확인) 뷰 → 우선순위·담당자·시작일·마감일 중 하나라도 비어있는 항목이 표시돼요.
주기적으로 확인해서 빈값을 채워주세요.

예시. 마케팅 관련 좋은 아티클을 발견했어요. 팀에 공유하고 나중에 다시 찾기도 쉽게 하고 싶어요.
🔍 공유자료실에 올려두면 팀 전체가 분야별·유형별로 언제든 찾을 수 있어요.
1

공유자료실 섹션 → 자료 추가 버튼 클릭
파일/링크·한줄요약·분야·자료유형·사업부를 입력해요. 한줄요약을 잘 써두면 나중에 찾기 훨씬 편해요.

2

자료를 찾을 때는 목적에 맞는 뷰를 골라요
사업부별 BD·LB·Tech 기준  ·  분야별 마케팅·영업 등  ·  자료유형별 아티클·레퍼런스·영상 등

3

자주 참고하는 자료는 즐겨찾기에 나를 추가 → 내 즐겨찾기 뷰에서 바로 접근할 수 있어요.
