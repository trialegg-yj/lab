향 언어화 엔진
적용 범위: 전 유형 공통 — 임팩트 최대
공통 페인포인트 해당 유형
"따뜻한 향" "숲 같은 향" → 주관적, 조향사마다 다르게 해석조향사, 제조사

솔루션 방향 — 자연어 감성 키워드 → 향 노트/계열 매핑 모델 · 브랜드 무드보드 → 향 프로파일 자동 생성 · 소비자 리뷰 텍스트 → 향 특성 구조화
AI 개선 포인트: 향 감성어 코퍼스 구축이 핵심. 한국 소비자 맥락 특화 필요 (향따 커뮤니티 용어 포함)
P2
저비용 향 프로토타이핑
적용 범위: 신생 + 소규모 브랜드 — 진입장벽 해소
공통 페인포인트 해당 유형
조향사 의뢰 비용 높아 방향 자체를 못 잡음신생, 소규모
소량 의뢰 시 조향사 상담 효율 낮음조향사
MOQ 강제로 실패 리스크 전가 (폐업 브랜드 교훈)폐업 케이스
솔루션 방향 — 조향사 의뢰 전 AI로 후보 향 방향 2~3개 제시 → 의사결정 비용 절감 · 소량 샘플링 단계에서 AI 필터링 → 수정 횟수 감소
협업 포지셔닝: 조향사와 경쟁이 아닌 사전 필터 → 정교화 구조로 접근
P3
소비자 향 선호 데이터 피드백 루프
적용 범위: 소규모 + 규모있는 브랜드
공통 페인포인트 해당 유형
데이터 없이 직관으로만 신제품 향 결정소규모, 규모있는

라인 확장 시 브랜드 향 일관성 유지 어려움규모있는 브랜드
솔루션 방향 — 판매 데이터 + 리뷰 텍스트 → 향 선호 패턴 도출 · 기존 히트 제품 향 프로파일 → 다음 라인 추천
AI 개선 포인트: 외부 커머스 데이터 연동 or 자체 수집 체계 설계가 선행 과제
P4
조향사 AI 브리프 구조화 도구
적용 범위: 조향사 ↔ 브랜드 사이 효율화
공통 페인포인트 해당 유형

소량 의뢰 클라이언트 응대 비용 높음조향사, 제조사
솔루션 방향 — 클라이언트가 AI로 요구사항 사전 정리 → 조향사에게 구조화된 브리프 전달 · "이런 향 원해요"를 노트/농도/계열로 자동 변환
협업 포인트: 조향사를 파트너로 온보딩하면 실제 향 개발 데이터 확보 → AI 모델 고도화 선순환
P5
제조사 연계 원스톱 패키지
적용 범위: 제조사 B2B — 파트너십 확장

제조 강하지만 향 기획 역량 없음제조사

B2C 체험 중심으로 선점 중. 우리가 B2B(소규모 브랜드 AI 조향 지원)에 집중하면 충돌 최소화. 데이터 축적·반복 재현성·브랜드 맞춤화 등 B2B 전용 기능을 차별점으로 강조.
🤝 perfumer.k 와의 관계 설정
경쟁보다 파트너십 가능성. 공장(perfumer.k) + AI 데이터(우리) 조합으로 풀스택 서비스 제공. 공장 역량은 없지만 AI로 개발 시간·비용을 줄이는 포지션 제안 가능.
🔗 오프라인 공방 연계 모델
오프라인 공방(아로마케미·아로마포레스트 등)은 AI 없는 체험만 제공. AI 추천 → 오프라인 체험 → 제품화 파이프라인 구성 시 상호 보완 파트너 가능.
📊 시장 수요 확인
오프라인 공방 성장 + 북촌 향수 성지 바이럴 = '나만의 향' 수요는 실재한다. 이 수요를 온라인·AI로 접근하는 것이 우리의 기회. 오프라인 한계(지역·시간·가격)를 해소.
